# javafx

### Тестовое приложение делалось по этой статье
https://openjfx.io/openjfx-docs/#maven

### Генератор проекта 
mvn archetype:generate -DarchetypeGroupId=org.openjfx -DarchetypeArtifactId=javafx-archetype-simple -DarchetypeVersion=0.0.3 -Djavafx-version=23.0.2

### Запуск проекта
mvn clean javafx:run
